컴퓨터공학전공 19학번 김미경

IntelliJ 단축키 15가지

단축키 기능 비고
01 Alt + Enter 코드에서 발견된 오류 위에 커서를 놓고, Alt+Enter을 누르면 문제에 대한 수정 제안 목록이 표시된다.
02 F2 수정이 필요한 곳으로 이동한다.
03 MacOS: Cmd + 1
Win/Linux: Alt + 1 		프로젝트 창을 연다
04 Esc 모든 팝업창을 닫는다.
05 MacOS: Cmd + E
Win/Linux: Ctrl + E 		가장 최근 열었던 파일을 확인한다.
06 MacOS: Cmd + B
Win/Linux: Ctrl + B 		심볼이 구현된 클래스 파일로 이동한다. Alt를 함께 누르면, 모든 파일에서 선언된 위치를 탐색한다.
07 Alt + F7 모든 사용 위치가 표시된다. 선언을 검색하는 것과 다르다
08 Crtl 두번 클릭 어느 위치에서든 Run Anything 창이 열린다.
09 MacOS: Alt+↑ / Alt+↓
Win/Linux: Ctrl+W / Ctrl+Shift+W		 커서 주변의 코드 섹션(선택된 영역)을 확장하거나 축소한다.
10 MacOS: Cmd + /
Win/Linux: Ctrl + / 		간편하게 주석을 추가한다.
11 MacOS: Cmd+Shift+Enter
Win/Linux: Ctrl+Shift +Enter		 현재 구문을 완성한다. 중괄호를 자동으로 생성하고 다음 필요한 위치로 커서가 이동한다.
12 MacOS: Cmd + Alt + L
Win/Linux: Ctrl + Alt + L 		현재 파일의 서식을 프로젝트 표준에 맞춘다.
13 MacOS: Ctrl + T
Win/Linux: Ctrl+Shift+T 		모든 리펙토링을 표시한다.
14 MacOS: Cmd + Shift + A
Win/Linux: Ctrl+Shift+A 		액션 찾기 창이 열린다. 액션은 물론, 설정도 검색 가능해서, 이 창에서 설정을 바로 변경할 수 있다.
15 Shift 두 번 클릭 전체 검색 창이 표시된다. 클래스, 파일, 심볼 및 액션의 검색 결과가 모두 표시된다. 명령어도 지원한다. 기본적으로 최근 파일이 표시되므로 Cmd + E 대신 이 창을 활용할 수 있다.
About

IntelliJ에서 유용하게 사용되는 15가지의 기능들을 html을 이용해 정리하였습니다. 코드 에디터마다 사용되는 단축키들이 다르기 때문에 미리 단축키를 숙지해두면 코딩이 좀 더 간편해집니다. 한 학기 동안 IntelliJ를 이용해 프로젝트를 개발할 예정이므로 개발을 하면서 사용되는 단축기들을 추가해 나갈 계획입니다.
